# Синтаксис Markdown!

## Выделение текста
* Жирный текст -** **пример**
* Курсивный текст -* *пример* 
* Зачеркнутый текст -~~ ~~пример~~
* Заголовок -# в начале строки
## Списки
    * нумерованные - обычными цифрами
    * ненумерованные - * вначале строки
* Вложенные списки - нужен отступ
