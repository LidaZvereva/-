# Синтаксис Markdown!

## Выделение текста
* Жирный текст -** **пример** или __пример__
* Курсивный текст -* *пример* или _пример_
* Зачеркнутый текст -~~ ~~пример~~
* Заголовок -# в начале строки
## Списки
    * нумерованные - нумеруем 1,2,3
    * ненумерованные - добавляем (*)
* Вложенные списки - нужен отступ
